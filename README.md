# SQL Practice Repository

This repository contains my SQL practice programs and database exercises developed while learning Database Management Systems and MySQL concepts. The repository includes basic to intermediate SQL programs such as database creation, table creation, inserting records, updating data, deleting records, filtering data using conditions, sorting results, joins, aggregate functions, and constraints.

Through these exercises, I improved my understanding of relational databases, query writing, and structured data management. This repository reflects my continuous learning journey in SQL and backend technologies.

- DML Commands
